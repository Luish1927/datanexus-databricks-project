## Como manter
1. Adicionar uma entrada em `[Unreleased]` a cada PR/alteração relevante.
2. No release, mover itens para uma seção datada (`YYYY-MM-DD`).
3. Categorizar sempre em `Added`, `Changed`, `Fixed` ou `Removed`.
4. Referenciar arquivos/notebooks impactados.
